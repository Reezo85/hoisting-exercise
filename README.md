# Hoisting Exercise

Student: Reginald Boyd  
Course: Springboard Software Engineering Career Track

## Purpose
Practice understanding how JavaScript hoisting works with:
- `var`
- `let`
- `const`

## Files
- `hoisting_exercise_starter.js` — Starter tasks
- `hoisting_exercise_solution.js` — Completed solution
- `index.html` — Loads the exercise in the browser

## How to Run
1. Open `index.html` in a browser  
2. Open the console (Ctrl+Shift+J / Cmd+Option+J)  
3. View logged outputs and hoisting behavior
